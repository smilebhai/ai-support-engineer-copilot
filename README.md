# AI Support Engineer Copilot

A LangGraph and LangSmith workflow for automating support issue triage and escalation preparation.

## Features

- Issue classification using GPT-4o-mini
- Conditional routing with LangGraph
- Category-specific troubleshooting workflows
- Escalation summary generation
- LangSmith tracing and evaluation
